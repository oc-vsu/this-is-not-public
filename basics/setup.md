# Setup

## Installation

## GOROOT & GOPATH

- GOROOT ist der Installationspfad unter dem GO installiert ist:
    ```
    $ go env GOROOT
    /usr/local/go
    ```
- GOPATH definiert den Workspace unter dem GO Projekte erstellt werden. I.d.R im home-Verzeichnis:
    ```
    $ go env GOPATH
    /Users/vsu/go
    ```
